install Angular Cli, node.js, express.js and import the data into three databases
1. start the MySQL, neo4j services
2. Back-end, go to the adb_backend folder, use 'node app.js' to start the back-end service
3. Front-end, go to the adb_frontend folder, use 'ng serve' to start, then open browser 'http://localhost:4200'.
